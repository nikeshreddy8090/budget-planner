# Real-Time Budget Planner

Project Overview

The **Real-Time Budget Planner** is a web-based application designed to help users efficiently manage their income, expenses, and savings. It provides real-time updates of financial transactions, allowing users to track their spending habits and maintain a balanced budget. The application offers an intuitive interface for adding, editing, and deleting transactions while automatically calculating the remaining balance.

Objectives

* Track income and expenses in real time.
* Calculate the available balance automatically.
* Categorize financial transactions.
* Help users plan and monitor their monthly budget.
* Provide a simple and user-friendly interface for personal finance management.

Features

* ➕ Add income and expense transactions.
* ✏️ Edit and delete existing transactions.
* 💰 Automatic balance calculation.
* 📊 Display total income, total expenses, and remaining balance.
* 📅 Real-time transaction updates.
* 📱 Responsive design for desktop and mobile devices.
* 💾 Local storage support to retain data after page refresh.

Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript
* **Storage:** Browser Local Storage
* **IDE:** Visual Studio Code
